## 📡 Mensageria na AWS (alertas de sensores)

Serviço simples de mensageria na nuvem que dispara **alerta por e-mail**
(via **Amazon SNS**) quando uma **leitura de sensor da Fase 3** sai da faixa
ideal de cultivo.

➡️ **Solução completa, passo a passo e prints:** [AWS_Mensageria/README.md](AWS_Mensageria/README.md)

```
Sensores (Fase 3)  ->  Python (boto3 + regras)  ->  Amazon SNS  ->  E-mail do produtor
```
